# こもれび平原 - Komorebi Plains

ブラウザで遊べる 3D オープンワールド探索ゲーム。
プロシージャル生成のはてしない地形を歩き回り、ひかるクリスタルを集めよう。

## 遊び方

| 操作 | PC | スマホ |
|---|---|---|
| 移動 | WASD / 矢印キー | 左下ジョイスティック |
| 視点 | マウスドラッグ (クリックでポインターロック) | 画面ドラッグ |
| ジャンプ | Space | 右下ボタン |
| ダッシュ | Shift | スティックを強く倒す |

- 💎 クリスタルに近づくと自動で取得
- ☀️ 4分で1日が過ぎる昼夜サイクル。夜は星空とクリスタルの輝きが見どころ
- 🐇 野ウサギは近づくと逃げる

## 特徴

- **無限地形**: ノイズベースのプロシージャル生成。チャンク式ロードでどこまでも歩ける
- **アセットゼロ**: 画像・モデル・音源ファイルなし。
  形状は Three.js プリミティブ、効果音は WebAudio 合成
- **ビルド不要**: ES Modules + importmap。`npx serve` だけで動く

## ローカルで動かす

```bash
npx serve -l 8000 .
# または: python3 -m http.server 8000
# ブラウザで http://localhost:8000
```

## 技術スタック

- Three.js 0.160 (CDN ESM / importmap)
- Vanilla JavaScript (ES Modules)
- WebAudio API (効果音合成)
